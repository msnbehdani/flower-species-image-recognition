\##Project Description

This project builds a Convolutional Neural Network (CNN) to classify flower species from images.



The model is trained using the TensorFlow Flowers dataset and can classify images into five flower categories.





\##Flower Classes

daisy

dandelion

roses

sunflowers

tulips

Dataset





\##Dataset source:



https://www.tensorflow.org/datasets/catalog/tf\_flowers



Download:



http://download.tensorflow.org/example\_images/flower\_photos.tgz





\##Technologies Used

Python

TensorFlow

Keras

NumPy

Matplotlib

Scikit-learn
